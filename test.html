iOS

iOS是一個由蘋果公司開發和發布的手機操作系統。 最初是於2007年首次發布iPhone、iPod Touch和Apple TV。 iOS派生自OS X，它們共享Darwin基礎。 OS X操作系統是用在蘋果電腦上，iOS是蘋果的移動版本。

SVN

Apache Subversion，通常縮寫為SVN，是一款開源的版本控制系統軟件。 Subversion由CollabNet公司在2000年創建。 但是現在它已經發展為Apache Software Foundation的一個項目，因此擁有豐富的開發人員和用戶社區。

jMeter

jMeter是一款開源的測試軟件。 它是100%純Java應用程序，用於負載和性能測試。

EJB

Enterprise Java Beans（EJB）是一個創建高度可擴展性和強大企業級應用程序的開發架構，部署在兼容應用程序服務器（比如JBOSS、Web Logic等）的J2EE上。

Spring

Spring框架是一個開源的Java平台，為快速開發功能強大的Java應用程序提供了完備的基礎設施支持。

Spring框架最初是由Rod Johnson編寫的，在2003年6月首次發佈於Apache 2.0許可證下。
